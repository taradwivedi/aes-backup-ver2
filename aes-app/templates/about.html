About the Automated Essay Scoring System

What is an AES System?

An Automated Essay Scoring (AES) System is an advanced software tool that evaluates and scores written essays using artificial intelligence and natural language processing. AES systems are designed to provide instant, objective, and consistent feedback to writers, helping them improve their writing skills and saving educators significant time.

Features of Our AES System

  • Grammar & Spelling Check: Instantly highlights grammar issues and spelling mistakes.
  • Score Prediction: Uses a trained LSTM neural network to predict essay scores on a 10-point scale.
  • Detailed Feedback: Provides actionable suggestions for improvement, including vocabulary and structure tips.
  • Advanced Vocabulary Detection: Recognizes and rewards the use of sophisticated words and phrases.
  • Downloadable Reports: Easily download your evaluation as a PDF or DOCX file.
  • Modern UI: Clean, responsive interface with light/dark mode support for comfortable use.
  • File Upload: Upload DOC/DOCX files for instant analysis.

How Does It Work?

Our AES system leverages a Long Short-Term Memory (LSTM) neural network—a type of deep learning model well-suited for understanding sequences of text. The process involves:

  1. Preprocessing: Cleans and normalizes the essay text.
  2. Feature Extraction: Calculates readability scores, word counts, and other linguistic features.
  3. Tokenization: Converts text into sequences for the LSTM model.
  4. Scoring: The LSTM model predicts a score based on both the essay content and extracted features.
  5. Grammar Feedback: Uses LanguageTool to highlight grammar and spelling issues, providing color-coded feedback.
  6. Report Generation: Allows users to download a comprehensive evaluation report.

This combination of machine learning and rule-based feedback ensures both accuracy and actionable insights for writers.

Why Was This System Made?

This AES system was developed as a final year project by four dedicated students (Tarangini Dwivedi, Nandana Nambiar, Sejal Jadhav, Neeti Jample) with a passion for artificial intelligence and educational technology. Our goal was to tackle the complex challenge of automating essay evaluation—a task that requires both linguistic understanding and robust machine learning. By combining deep learning with user-friendly design, we aim to empower students, teachers, and lifelong learners to improve their writing with instant, reliable feedback.
